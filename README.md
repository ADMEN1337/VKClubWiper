# VKClubWiper

Бот для приложения ["В Клубе"](http://vk.com/app4397521) для [2ch](2ch.hk)

### Запуск

```sh
python ./bot.py
```

### Команды

```sh
h - вывод справки.
c - подключение всех аккаунтов
g - перекатиться в клуб
m MES - отправить MES в чат
q - выход
d - дизлайк
l - лайк
L - рога (без репостов и прочей хуйни)
e - кинуть яйцо
```
